# TextBuddy — Social Media Post Artifact Skill

## Artifact Type
Social media post — a single, platform-optimized post ready to publish.

## Template Structure

Generate the post with these sections in order:

### 1. Hook Line
- Open with a statement that stops the scroll in the first 5-10 words.
- Techniques: bold claim, surprising stat, provocative question, contrarian take.
- The hook must be self-contained — it appears above the "see more" fold.
- Do not start with "We are excited to announce" or "Check out our latest."
- Cite evidence for any factual hook: statistics, results, or data points.

### 2. Body
- Expand on the hook with supporting detail, story, or argument.
- Structure for scannability:
  - Short paragraphs (1-2 sentences each).
  - Line breaks between paragraphs for visual breathing room.
  - Bullet points or numbered lists for multiple items.
- Use evidence findings to back claims — cite as `[source:ID]`.
- Write in the brand voice. Match the energy, vocabulary, and personality exactly.
- Include a human element where possible: lessons learned, behind-the-scenes, team wins.

### 3. Hashtags
- Place hashtags at the end of the post, separated by a line break.
- Use 3-5 hashtags for LinkedIn. Use 1-3 for Twitter/X. Use 5-10 for Instagram.
- Mix reach hashtags (broad, high-volume) with niche hashtags (specific, targeted).
- Use camelCase for multi-word hashtags for accessibility: #ContentMarketing not #contentmarketing.
- Never use banned, irrelevant, or spammy hashtags.

### 4. Call to Action (CTA)
- End with one clear engagement prompt before the hashtags.
- Match CTA to the post's goal:
  - Engagement: "What's your take? Drop it below."
  - Traffic: "Link in the comments [or bio]."
  - Sharing: "Tag someone who needs to see this."
  - Conversion: "DM us [keyword] to get started."
- Keep the CTA to one sentence. Make the action effortless.

## Platform-Specific Rules

### Twitter / X
- **Maximum**: 280 characters. Write tight — every word must earn its place.
- No hashtag block — weave 1-2 hashtags into the text naturally.
- Threads: if content exceeds 280 chars, split into a thread with 1/ 2/ 3/ numbering.
  Each tweet must stand alone and hook the reader into the next.
- Do not include links in the main tweet body if possible — add in a reply.

### LinkedIn
- **Maximum**: 3,000 characters. Aim for 800-1,500 for optimal engagement.
- Open with a strong hook — only the first 2-3 lines show before "see more."
- Use line breaks aggressively. Dense paragraphs kill LinkedIn engagement.
- Professional but human tone. Personal stories outperform corporate announcements.
- End with a question to drive comments.

### Instagram
- **Maximum**: 2,200 characters for captions.
- Write a compelling first line — only 125 characters show in the feed.
- Hashtags go in the first comment if brand voice prefers clean captions.
- Use emoji per brand voice guidelines — do not add emoji if brand voice does not include them.
- Structure: hook, story/value, CTA, hashtags.

## Emoji Usage

- Follow the brand voice guidelines for emoji frequency and style.
- If brand voice allows emoji: use 1-3 per post, placed at line beginnings or as bullet markers.
- If brand voice prohibits emoji: use none. Zero. Not even common ones.
- Never use emoji as a substitute for clear writing.
- Avoid these in professional B2B content: fire, rocket, 100, eyes.

## Engagement Optimization

- Ask questions to invite comments.
- Use "you" and "your" to speak directly to the reader.
- Share specific numbers over vague claims: "37% increase" beats "significant growth."
- Controversy (thoughtful, not inflammatory) drives engagement — take a clear position.
- Time-sensitive hooks ("This week," "Just discovered," "New data shows") create urgency.

## Rules

- Write one post per output unless HITL requests a batch or thread.
- Never fabricate metrics or results — all numbers must cite evidence.
- Adapt length and tone to the platform specified in HITL answers.
- If no platform is specified, default to LinkedIn format.
- Do not include image descriptions or alt-text unless explicitly requested.
- If HITL answers reference a product launch, event, or campaign, align the post to that context.
