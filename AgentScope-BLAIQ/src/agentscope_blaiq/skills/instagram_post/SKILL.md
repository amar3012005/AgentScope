### Instagram Post Blueprint

1. **Hook (first line):** Start with a sharp, scroll-stopping line.
2. **Core value:** Explain one clear insight or benefit in plain language.
3. **Brand framing:** Keep Solvis tone: intelligent, modular, future-proof, human.
4. **Structure:** Use short lines, 1-2 emoji max only when appropriate.
5. **CTA:** End with one action (comment, DM, visit link in bio).
6. **Hashtags:** 5-10 relevant tags, mixing brand and topical terms.

### Format

- Output in clean markdown.
- Keep post body concise (about 80-180 words unless user asks otherwise).
- Add a separate `Hashtags:` line at the end.
