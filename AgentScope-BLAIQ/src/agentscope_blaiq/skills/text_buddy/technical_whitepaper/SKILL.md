# Technical Whitepaper Skill

## Objective
Create authoritative, standards-compliant technical documentation for industrial systems, products, and processes.

## Tone
Formal, precise, and technically rigorous with clear engineering terminology.

## Core Sections
1. Executive Summary
2. Technical Overview
3. Methodology
4. Performance Metrics
5. Safety Compliance
6. Case Study Implementation
7. Conclusion

## Evidence Protocols
- Must include IEEE/ISO standard references
- Requires technical specifications tables
- Must cite peer-reviewed engineering studies

## Checklist
- [ ] System architecture diagrams
- [ ] Performance validation data
- [ ] Risk assessment sections
- [ ] API/Interface documentation
- [ ] Maintenance procedure guidelines
