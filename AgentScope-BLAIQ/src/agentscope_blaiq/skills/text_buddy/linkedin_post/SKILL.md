# TextBuddy — LinkedIn Post Artifact Skill

## Artifact Type
LinkedIn post — a single, scroll-stopping social post optimized for the LinkedIn feed algorithm.

## Template Structure

Generate the post with these sections in order:

### 1. Hook Line
- 1-2 lines maximum that stop the scroll.
- Techniques: bold claim, surprising stat, contrarian take, personal lesson.
- Must appear above the "see more" fold (first 2-3 lines, ~200 characters).
- Never start with "We are excited to announce" or "Check out our latest."
- Cite evidence for any factual claim: `[source:ID]`.

### 2. Context / Body
- Expand on the hook with supporting detail, story, or argument.
- Use short paragraphs (1-2 sentences each) with line breaks between them.
- Dense paragraphs kill LinkedIn engagement — break aggressively.
- Use bullet points for 3+ items or key specs.
- Write in the brand voice — match energy, vocabulary, and personality exactly.
- Include a human element: lessons learned, behind-the-scenes, team wins.

### 3. Value Pillars
- Present 3 key benefits, specs, or takeaways as bullet points.
- Each bullet: specific, concrete, no fluff.
- Example: "Modular design — scale from 5 kW to 50 kW without replacing the core unit."

### 4. Integration / Proof
- Mention relevant integrations, partnerships, or proof points.
- Cite evidence: case studies, benchmarks, or proven results `[source:ID]`.
- Keep to 1-2 sentences — this supports the main argument, it doesn't replace it.

### 5. Call to Action (CTA)
- End with one clear engagement prompt.
- Match CTA to goal:
  - Engagement: "What's your take? Drop it below."
  - Traffic: "Link in the comments."
  - Sharing: "Tag someone who needs to see this."
  - Conversion: "DM us to get started."
- Keep to one sentence. Make the action effortless.

### 6. Hashtags
- Place at the end, separated by a blank line.
- Use 3-5 hashtags. Mix broad (#RenewableEnergy) with niche (#HeatPumpTech).
- Use camelCase for accessibility: #RenewableEnergy not #renewableenergy.
- Never use banned, irrelevant, or spammy hashtags.

## Formatting Rules

- Output as clean markdown — no HTML, no code blocks.
- Target 800-1,500 characters for optimal engagement.
- Use line breaks between every paragraph — no dense blocks of text.
- Bold key phrases sparingly for emphasis (1-3 per post).
- Emoji: follow brand voice. If allowed, use 1-3 max. If prohibited, use zero.
- Hashtags go on their own line at the end.

## Tone Rules

- Professional but human — personal stories outperform corporate announcements.
- Use "you" and "your" to speak directly to the reader.
- Confident but not arrogant — present solutions as proven, not as favors.
- No exclamation marks unless brand voice explicitly allows them.

## Rules

- Write one post per output unless HITL requests a batch.
- Never fabricate metrics or results — all numbers must cite evidence.
- If no platform is specified, default to this LinkedIn format.
- Do not include image descriptions or alt-text unless explicitly requested.
- If HITL references a product launch, event, or campaign, align the post to that context.
