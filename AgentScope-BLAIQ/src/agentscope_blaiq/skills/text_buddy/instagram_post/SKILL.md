# TextBuddy — Instagram Post Artifact Skill

## Artifact Type
Instagram post — a visually-driven caption optimized for the Instagram feed algorithm.

## Template Structure

Generate the post with these sections in order:

### 1. Hook (First Line)
- Start with a sharp, scroll-stopping line.
- Only the first 125 characters show in the feed before "… more."
- Techniques: bold claim, surprising stat, provocative question, visual cue.
- Never start with "We are excited to announce" or generic corporate speak.
- Cite evidence for any factual claim: `[source:ID]`.

### 2. Core Value
- Explain one clear insight, benefit, or story in plain language.
- Keep it to 2-3 short paragraphs (1-2 sentences each).
- Use line breaks between paragraphs for visual breathing room.
- Write in the brand voice — match energy, vocabulary, and personality exactly.

### 3. Brand Framing
- Weave in brand attributes naturally: intelligent, modular, future-proof, human.
- Do not list features — tell a story about the benefit.
- Keep to 1-2 sentences. This supports the main argument, it doesn't replace it.

### 4. Call to Action (CTA)
- End with one clear action prompt.
- Match CTA to goal:
  - Engagement: "What's your take? Drop it below."
  - Traffic: "Link in bio."
  - Sharing: "Tag someone who needs to see this."
  - Conversion: "DM us [keyword] to get started."
- Keep to one sentence. Make the action effortless.

### 5. Hashtags
- Place at the end, separated by a blank line.
- Use 5-10 hashtags. Mix brand (#Solvis) with topical (#RenewableEnergy, #HeatPump).
- Use camelCase for accessibility: #RenewableEnergy not #renewableenergy.
- Never use banned, irrelevant, or spammy hashtags.

## Formatting Rules

- Output as clean markdown — no HTML, no code blocks.
- Keep post body concise: 80-180 words unless user asks otherwise.
- Use short lines and line breaks — Instagram captions are scanned, not read.
- Emoji: follow brand voice. If allowed, use 1-2 max. If prohibited, use zero.
- Add a separate `Hashtags:` line at the end.

## Tone Rules

- Conversational and visual — write as if accompanying a strong image.
- Use "you" and "your" to speak directly to the reader.
- Confident but approachable — no corporate stiffness.
- No exclamation marks unless brand voice explicitly allows them.

## Rules

- Write one post per output unless HITL requests a batch.
- Never fabricate metrics or results — all numbers must cite evidence.
- Do not include image descriptions or alt-text unless explicitly requested.
- If HITL references a product launch, event, or campaign, align the post to that context.
