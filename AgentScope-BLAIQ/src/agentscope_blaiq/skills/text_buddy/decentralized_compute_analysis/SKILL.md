## Decentralized Compute Market Analysis Skill

**Purpose:**
Generate structured competitive analysis comparing decentralized compute platforms using cost-per-token efficiency as the primary metric.

**Constraints:**
- Must define cost-per-token efficiency as (computational power per dollar) × (network reliability score) ÷ (token volatility index)
- Include at least 3 competing platforms in each analysis
- Provide benchmark comparisons against AWS/GCP pricing models

**Methodology:**
1. Calculate token cost efficiency ratios
2. Map network performance metrics
3. Generate comparative matrices
4. Highlight strategic advantages/disadvantages

**Examples:**
```markdown
[Platform X] 0.82¢/token vs [Platform Y] 1.15¢/token
AWS equivalent: 1.5¢/token (35% less efficient)
```
