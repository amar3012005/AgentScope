# out-of-box-agents-map

This note organizes the prebuilt agents in the docs by use case.

## Covered docs

- [Alias](https://docs.agentscope.io/out-of-box-agents/alias)
- [Finance Analysis](https://docs.agentscope.io/out-of-box-agents/alias-finance)
- [Browser-use Agent](https://docs.agentscope.io/out-of-box-agents/browser-use)
- [Data Science](https://docs.agentscope.io/out-of-box-agents/data-science)
- [DataJuicer Agent](https://docs.agentscope.io/out-of-box-agents/datajuicer-agent)
- [Deep Research](https://docs.agentscope.io/out-of-box-agents/deep-research)
- [EvoTraders](https://docs.agentscope.io/out-of-box-agents/evo-trader)

## Main themes

- Deep Research is the best example of decomposition, deep search, reflection, and report generation.
- Alias is the general-purpose base agent with specialized modes.
- Browser-use focuses on task completion through browser automation.
- Data Science and DataJuicer show task-specific multi-step systems.
- EvoTraders shows a self-evolving domain-specific multi-agent setup.

## Practical reading order

1. Read Deep Research first if you want the most architecture-rich example.
2. Read Alias and Finance Analysis for general-purpose and evidence-heavy patterns.
3. Read Browser-use, Data Science, DataJuicer, and EvoTraders for task-specific system design.
